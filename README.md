# E-Commerce Web App (Local)

This repository was generated by a script.

Structure:
- client/ — React + Vite frontend
- server/ — Node.js + Express backend

Setup (local):

1. Ensure Node.js (>=16) and npm are installed.
2. Ensure MongoDB is running locally, or use an Atlas URI.
3. Copy .env.example to server/.env and fill values (MONGO_URI, JWT_SECRET).
   Also copy .env.example to client/.env (or set VITE_API_URL in the client's .env).
4. From project root run:

   # install dependencies (the script may have already done this)
   cd server
   npm install

   cd ../client
   npm install

5. Seed sample products (from project root):
   cd server
   node src/seed.js

6. Start backend and frontend (two terminals):
   # Backend
   cd server
   npm run dev

   # Frontend
   cd client
   npm run dev


Notes:
- Authentication: JWT
- Default roles: 'user' and 'admin'. Create an admin user via Register endpoint and then change role in DB, or add admin creation logic.
